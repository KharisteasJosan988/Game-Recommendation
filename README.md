# Game Recommendation System

A content-based filtering system that recommends games based on user preferences using Natural Language Processing techniques.

---

##  Overview
This project builds a game recommendation system using **TF-IDF Vectorization** and **Cosine Similarity** to find similar games based on their descriptions and features.

---

## Dataset
- Contains **27,000+ game records**
- Features include:
  - Game title
  - Genre
  - Description
  - Tags

---

## Technologies Used
- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorizer
- Cosine Similarity
- Jupyter Notebook

---

## Methodology
1. Data Cleaning & Preprocessing
2. Text Vectorization using TF-IDF
3. Similarity Calculation using Cosine Similarity
4. Recommendation based on similarity scores

---

## Evaluation
- Evaluated using:
  - Precision@K
- Produces relevant recommendations based on content similarity
